# OpenClaw小龙虾培训

使用 Reveal.js 制作的网页幻灯片，主题为 OpenClaw 个人 AI 助手平台培训。

## 内容大纲

1. 什么是小龙虾？
2. 背景故事与诞生
3. 当前现状与生态
4. 保姆级安装教程
5. 风险提示与安全建议
6. 未来展望

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

## 部署到 Gitee Pages

1. 在 Gitee 上创建新仓库
2. 将本文件夹内容推送到 Gitee 仓库
3. 进入仓库 → 服务 → Gitee Pages
4. 点击「启动」
5. 等待构建完成，获得在线访问地址
