四、保姆级安装教程

环境要求

操作系统要求
macOSNode ≥22
LinuxNode ≥22
WindowsWSL2（强烈推荐）

安装步骤

# 1. 安装OpenClaw
npm install -g openclaw@latest

# 2. 启动安装向导
openclaw onboard --install-daemon

# 3. 启动Gateway
openclaw gateway --port 18789 --verbose

# 4. 发送测试消息
openclaw message send --to +1234567890 --message "Hello"

飞书配置

  1. 在飞书开放平台创建应用
  2. 获取 App ID 和 App Secret
  3. 配置权限
  4. 在OpenClaw配置中添加
{
  "channels": {
    "feishu": {
      "enabled": true,
      "appId": "YOUR_APP_ID",
      "appSecret": "YOUR_APP_SECRET"
    }
  }
}

验证安装

# 运行诊断
openclaw doctor

# 检查状态
openclaw status

五、风险提示与安全建议

策略说明
